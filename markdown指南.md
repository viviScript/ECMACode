### 1.5. 链接
#### 内链式
[百度1](http://www.baidu.com/"百度一下"){:target="_blank"}   
```
[百度1](http://www.baidu.com/"百度一下"){:target="_blank"}   
```
#### 引用式
[百度2][2]{:target="_blank"}
[2]: http://www.baidu.com/  "百度二下"
```[百度2][2]{:target="_blank"}
[2]: http://www.baidu.com/  "百度二下"
```
#### 邮箱链接式

```

```
### 1.6 图片
#### 内链式
![name](./01.png '描述')
```
![name](./01.png '描述')
```
#### 引用式
    ![name][01]
[01]: ./01.png '描述'
#### 图片带有链接式
    ![name](./01.png '百度')](http://www.baidu.com){:target="_blank"}   

[![name](./01.png '百度')][5]{:target="_blank"} 
[5]: http://www.baidu.com
